# PojavLauncherTeam

**PojavLauncherTeam** started years ago with one major goal: bringing Minecraft: Java Edition and its latest releases to even more users with the computers they carry around in their pockets. They, as a team, accomplished this goal... but it didn't come without controversy or hits to their name. As a result of this, the team decided that the best thing to do was to start over.
As such, **PojavLauncherTeam** and all of its assets are now archived. The team doesn't want to hide their past, but instead learn from it to grow in the future. Below are some links to successor software and resources:

- [TeamPojavLauncher](https://github.com/TeamPojavLauncher), the successor organization  
- [PojavLauncher Android](https://github.com/TeamPojavLauncher/PojavLauncher), the successor to the Android port.
- A new Discord server is available [here](https://discord.gg/2HYpzs4gZT).
