# PojavLauncherTeam

PojavLauncherTeam started years ago with one major goal: bringing Minecraft: Java Edition and its latest releases to even more users with the computers they carry around in their pockets. We, as a team, accomplished this goal... but it didn't come without controversy or hits to our name. As a result of this, we decided that the best thing to do was to start over.

As such, **PojavLauncherTeam** and all of its assets are now archived. We don't want to hide our past, but instead learn from it to grow in the future. Below are some links to successor software and resources:

- [AngelAuraMC](https://github.com/AngelAuraMC), the successor organization  
- [Amethyst iOS](https://github.com/AngelAuraMC/Amethyst-iOS), the successor to the iOS port.
- [Amethyst Android](https://github.com/AngelAuraMC/Amethyst-Android), the successor to the Android port.
- A new Discord server is available [here](https://discord.gg/5ptqkyZxEy).
